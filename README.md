# Fake Job Offer Detector

A Flask-based cybersecurity web application that detects suspicious job offers using scam keyword analysis, risk scoring, recruiter email verification, and PDF report generation.

## Features

- Scam Keyword Detection
- Risk Score Calculation
- Recruiter Email Verification
- Dynamic Recommendations
- PDF Report Generation

## Technologies Used

- Python
- Flask
- Bootstrap
- ReportLab

## How It Works

1. User enters a job offer and recruiter email.
2. The system analyzes scam-related keywords.
3. A risk score is generated.
4. Recruiter email is verified.
5. Security recommendations are displayed.
6. A PDF report can be downloaded.
